About Me

Perkenalkan nama saya Rachmat Farid Abdillah. merupakan mahasiswa Program Studi S1 Teknik Informatika, Fakultas Teknik, di Universitas Riau. Saya lahir di Bangkinang kota pada tanggal 14 Juli 2002. Saya mempunyai satu adik laki-laki dan satu kakak perempuan. Saya merupakan anak ke dua dari tiga bersaudara . Saya mempunyai beberapa hoby yaitu seperti menonton fim, membaca komik, dan olahraga. Ketika ada waktu luang saya biasanya membaca komik dan menonton film. Hal ini saya lakukan untuk menghilangkan rasa bosan. Saya juga suka berolahraga. Saya biasanya jogging satu kali dalam satu minggu. Awal mula saya menempuh pendidikan yaitu di Sekolah dasar islam terpadu insan cendekia Bangkinang. Saya menjalani pendidikan disana selama 6 tahun. Kemudian saya melanjutkan pendidikan ke SMP Negeri 1 Bangkinang kota selama 3 tahun. Lalu saya melanjutkan pendidikan di SMA Negeri 1 Bangkinang kota. Setelah lulus SMA saya mengikuti SBMPTN agar dapat melanjutkan pendidikan di perguruan tinggi. Alhamdullillah saya lulus masuk Program Studi S1 Teknik Informatika, Fakultas Teknik, di Universitas Riau. Saya ingin menjadi seorang software Engineer. Saya ingin merancang dan membuat berbagai aplikasi yang dapat digunakan untuk membantu berbagai kegiatan manusia. selain itu aplikasi ini nantinya juga akan saya rilis di beerbagai OS sehingga saya juga mendapatkan keuntungan. Oleh karena itu, selama menempuh pendidikan perguruan tinggi ini, saya akan belajar dengan sungguh-sungguh. Saya akan berusaha mempelajari segala hal yang nantinya akan saya perlukan ketika sudah masuk ke dunia kerja.
